1. Tactical Survival

Shoot Or Shield is a survival duel that continues indefinitely until one player dies.

The goal isn't just to kill early; it's to force the opponent into wasting shields and idles to accumulate match points, then deliver the final strike to claim the entire pot!

VS → KILL

2. Match Rules

In every round, both players simultaneously select one of the following moves:

  • SHOOT : Costs 1 resource unit. Eliminates an IDLE player.
  • SHIELD : Costs 1 resource unit. Deflects a SHOOT, scoring +1 match point.
  • IDLE : Free. Carries unit forward. Scores +1 match point if the opponent shielded.

3. Match Resource units

Match Resource units are chosen before each match using the menu slider. Both players begin with the same fixed number of units.

Every SHOOT or SHIELD costs 1 unit. When a player has no units left, only IDLE remains available.

In v3.1 this panel is now called Match Resource units, and this version uses a fixed pool for the whole match.

4. Matchup Matrix

vs KILL — shooter wins match
vs DEFLECT — shield scores +1pt
vs WASTE — idle scores +1pt
vs DUAL DEFEAT — both die
/ tie STANDOFF — no points

5. Flow & Timer

Every round has a 10-second selection phase. You can vote for any move during this time. The selection active at 0s is automatically committed.

A 5-second reveal phase shows both moves.

If the match ends in your victory, your profile inherits the combined match points of both players!
